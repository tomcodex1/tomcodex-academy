# AI Interviewer Milestone Roadmap

## Milestone 1: Interview Study Mode

Status: Complete

- Search Admin and Developer question banks.
- Filter by topic, source quality, and question type.
- Show live visible-result counts and clear all filters.

## Milestone 2: Spaced-Repetition Practice

Status: Complete

- Save difficult questions for review.
- Schedule weak questions more frequently.
- Track confidence and mastery by topic.

## Milestone 3: Realistic Interview Rounds

Status: Complete

- Provide screening, technical, scenario, and manager rounds.
- Add configurable round length and difficulty.
- Generate a round-by-round readiness report.

## Milestone 4: Developer Coding Workspace

Status: Complete

- Add Apex, SOQL, LWC, and debugging exercises.
- Validate solutions against expected concepts and test cases.
- Include governor-limit and security feedback.

## Milestone 5: Answer Comparison

Status: Complete

- Compare the candidate answer with a strong model answer.
- Highlight missing concepts, examples, and tradeoffs.
- Provide an improved answer structure.

## Milestone 6: Source Quality Score

Status: Complete

- Distinguish official, verified real-world, community, and generated content.
- Show source coverage and freshness.
- Prioritize high-confidence questions during study.

## Milestone 7: Personal Weakness Dashboard

Status: Complete

- Track performance by role, topic, and answer dimension.
- Recommend the next study session.
- Show readiness trends and milestone completion.
